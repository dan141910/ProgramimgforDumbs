USING CSS3 Gradients

a gradient (which is a blend between two or more color ) can be a nice background

CSS3 support two major types of gradients : linear and radial

syntax : background-image: linear-gradient(direction/angle, color/hex/rgb()/hsl()-start, color-stop, ...);

but, some browsers not support, so should write with webkit- and moz-

exam 1

background-image: linear-gradient(to right, red, yellow)

direction : to left, to bottom, to bottom right (chéo)...
angle : -90deg=left, [number]deg o

you also can repeat by code : repeating-linear-gradient()...
