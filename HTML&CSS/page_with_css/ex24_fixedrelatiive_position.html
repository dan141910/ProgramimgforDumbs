here is how to buiding a fixed menu

  • 1. Set the menu position to fixed by setting the position attribute to fixed.
  • 2. Give the menu a width with the width attribute.
  • 3.Consider the menu position by explicitly setting the top and left attributes.
  • 4. Set content position to relative.
  • 5. Change content’s left attribute to 20 percent.(100-menu)
  • 6.Give content a width property.

Delta

In reality, I rarely use absolute positioning for page layout. It’s just too difficult to get working and too inflexible for the range of modern browsers. However, it is still used in certain specialty situations like web game development where the programmer is deliberately subverting normal layout schemes for more control of the visual interface
